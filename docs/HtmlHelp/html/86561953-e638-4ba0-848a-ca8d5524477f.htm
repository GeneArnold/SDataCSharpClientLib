A AtomTextConstruct object that represents information that conveys a short summary, abstract, or excerpt for this entry.
Remarks

It is not advisable for theSummary property to duplicate Title or Content because Atom Processors might assume there is a useful summary when there is none.

Entries must contain a Summary in either of the following cases:

  1. The AtomEntry contains an Content property that has a Source property (and is thus empty).
  2. The AtomEntry contains content that is encoded in Base64; i.e., the ContentType property of Content property is a MIME media type, but is not an XML media type, does not begin with text/, and does not end with /xml or +xml.
