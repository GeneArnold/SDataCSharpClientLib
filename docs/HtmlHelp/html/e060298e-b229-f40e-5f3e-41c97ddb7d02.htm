Parameters
source (Uri)
A Uri that represents the URL of the syndication resource XML data.
credentials (ICredentials)
A ICredentials that provides the proper set of credentials to the source when required. This value can be null.
proxy (IWebProxy)
A IWebProxy that provides proxy access to the source when required. This value can be null.
settings (SyndicationResourceLoadSettings)
The SyndicationResourceLoadSettings object used to configure the AtomFeed instance. This value can be null.
Return Value
An AtomFeed object loaded using the source data.

ExceptionCondition
