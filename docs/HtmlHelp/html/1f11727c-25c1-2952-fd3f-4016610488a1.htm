The template resource allows a service consumer to obtain the resource's default property values. For example, the URL in the example could be used to fill the contents of a “Create Account” form. The $template segment is only valid after the resource kind segment (accounts in the example). Template resources are usually read only, but some applications could expose them in read/write mode, typically to admin accounts.
