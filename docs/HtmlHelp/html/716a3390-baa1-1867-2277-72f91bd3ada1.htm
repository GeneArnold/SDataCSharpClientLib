ItemsPerPage Property
Namespaces ► Sage.SData.Client.Core ► SDataResourceCollectionRequest ► ItemsPerPage
Number of resources that the service normally returns in a page. This number can be different from the value of the count parameter because the service provider is not required to cut pages exactly as requested by the consumer. It could impose a page size that is a multiple of some number. For example, or it may cap it. See query parameters section. There may also be a difference between this number and the number of resources in the current page. This is usually the case on the last page, which has less resources than “normal” pages.
