Value
A Uri that represents a base URI other than the base URI of the document or external entity. The default value is a null reference.
Remarks

The value of this property is interpreted as a URI Reference as defined in RFC 2396: Uniform Resource Identifiers, after processing according to XML Base, Section 3.1 (URI Reference Encoding and Escaping).
