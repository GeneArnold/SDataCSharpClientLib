A Uri that represents a Internationalized Resource Identifier (IRI) that identifies the remote location of this content.
Remarks

If a Source property is specified, the Content property must be empty. Atom Processors may use the IRI to retrieve the content and may choose to ignore remote content or to present it in a different manner than local content.

If a Source property is specified, the ContentTypeshould be provided and must be a MIME media type, rather than text, html, or xhtml. The value is advisory; that is to say, when the corresponding URI (mapped from an IRI, if necessary) is dereferenced, if the server providing that content also provides a media type, the server-provided media type is authoritative.

See RFC 3987: Internationalized Resource Identifiers for the IRI technical specification.

See System.Uri for enabling support for IRIs within Microsoft .NET framework applications.
