A AtomSource object that represents the meta-data of the source feed that this entry was copied from.
Remarks

The AtomSource is designed to allow the aggregation of entries from different feeds while retaining information about an entry's source feed. For this reason, Atom Processors that are performing such aggregation should include at least the required feed-level meta-data elements (id, title, and updated) in the AtomSource.
