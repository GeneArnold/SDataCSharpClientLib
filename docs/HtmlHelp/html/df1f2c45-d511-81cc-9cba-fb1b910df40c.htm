match (Predicate<(Of <(ISyndicationExtension>)>))
The Predicate<(Of <(T>)>) delegate that defines the conditions of the ISyndicationExtension to search for.
Return Value
The first syndication extension that matches the conditions defined by the specified predicate, if found; otherwise, the default value for ISyndicationExtension.
Remarks
The Predicate<(Of <(T>)>) is a delegate to a method that returns true if the object passed to it matches the conditions defined in the delegate. The elements of the current Extensions are individually passed to the Predicate<(Of <(T>)>) delegate, moving forward in the Extensions, starting with the first element and ending with the last element. Processing is stopped when a match is found.
