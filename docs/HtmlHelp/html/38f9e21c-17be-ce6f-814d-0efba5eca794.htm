Parameters
source (Uri)
A Uri that points to the location of the web resource used to load the syndication resource.
credentials (ICredentials)
A ICredentials that provides the proper set of credentials to the source resource when required. This value can be null.
proxy (IWebProxy)
A IWebProxy that provides proxy access to the source resource when required. This value can be null.
Remarks

  • If credentials is null, request is made using the default application credentials.
  • If proxy is null, request is made using the WebRequest default proxy settings.
  • After the load operation has successfully completed, the Loaded event will be raised.

Exceptions
ExceptionCondition
ArgumentNullExceptionThe source is a null reference (Nothing in Visual Basic).
FormatExceptionThe source data does not conform to the expected syndication content format. In this case, the feed remains empty.
XmlExceptionThere is a load or parse error in the XML. In this case, the feed remains empty.
