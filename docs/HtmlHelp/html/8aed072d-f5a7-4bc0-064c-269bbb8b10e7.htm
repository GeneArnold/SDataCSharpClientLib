Value
A DateTime that indicates the most recent instant in time when this entry was modified in a way the publisher considers significant. Publishers may change the value of this element over time. The default value is MinValue, which indicates that no update time was provided.
Remarks
The DateTime should be provided in Coordinated Universal Time (UTC).
